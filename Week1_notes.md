# week 1 notes

What is an API?
 API stands for Application Programming Interface,
 An API is a set of rules that allows two software applications to communicate with each other.



 What is JSON?
  JSON stands for Javascript Object Notation.
  JSON is lightweight data format used to store and exchange data between to systems.

 GET vs POST difference
  GET:
      is to request data from the server.
  POST:
      is to submit data to be processed to the server.
