# What is an API?
 API stands for Application Programming Interface,
 An API is a set of rules that allows two software applications to communicate with each other.


# Why should API keys never be uploaded?
 API keys act like digital passwords to your account; uploading them to public places allows anyone to steal them and access your private services.

# What happens if your API key leaks?
  An attacker can impersonate you, leading to stolen data, service disruptions, and massive unexpected bills charged to your account.

# What is JSON?
  JSON stands for Javascript Object Notation.
  JSON is lightweight data format used to store and exchange data between to systems.


# Difference between sending data and requesting data?
  Sending data (POST) is like mailing a letter to a server to update or create information, while requesting data (GET) is like asking the server to show you information it already has.
